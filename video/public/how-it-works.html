System Architecture

1. Frontend (Input)

The user pastes a URL. JavaScript validates the format and sends a POST request to our Node.js server using the fetch() API.

2. Backend (Processing)

Express receives the URL. It checks for keywords like "youtube" or "tiktok" to determine the origin. In a real app, this is where complex scraping logic lives.

3. Response (UI Update)

The server sends back a JSON object. JavaScript parses this object and dynamically updates the HTML DOM to show the video details.
